# sitkaz.kz — Ситуативный казахский (мини-апп)

Мини-приложение для изучения разговорного казахского по методике **«Ситуативный казахский»**.
Ядро обучения: **3 модуля, 20 уроков**, ситуативные фразы + место под видео-лекцию, практика (карточки), квиз и прогресс.

Работает и как обычный сайт в браузере, и как **Telegram Mini App** (подключён Telegram WebApp SDK).
Прогресс хранится локально в браузере (`localStorage`) — аккаунты и оплата пока не подключены.

## Структура проекта

```

│   ├── manifest.json   PWA-манифест
│   └── icon.svg        иконка
├── package.json
└── next.config.mjs
```

## Как наполнять контентом

Весь контент — в одном файле **`data/course.js`**.

**Добавить видео к уроку** — впишите ссылку в поле `video` (embed-ссылка YouTube/Vimeo):
```js
{ id: 1, module: "adam", title: "Танысу", ru: "Знакомство",
  video: "https://www.youtube.com/embed/XXXXXXXX",  // ← сюда
  phrases: [ ... ] }
```

**Добавить/поправить фразы** — редактируйте массив `phrases`:
```js
phrases: [
  { kk: "казахская фраза", ru: "перевод", tr: "произношение" },
]
```

Тексты фраз в 20 уроках — рабочая заготовка (структурный скелет). Замените их
реальными материалами курса Каната Тасибекова.

```bash
npm run build    # проверка продакшн-сборки
```

## Деплой: GitHub → Vercel → домен sitkaz.kz

1. **GitHub.** Создайте пустой репозиторий (напр. `sitkaz`). Из папки проекта:
   ```bash
   git init
   git add .
   git commit -m "sitkaz mini app — core"
   git branch -M main
   git remote add origin https://github.com/ВАШ_ЛОГИН/sitkaz.git
   git push -u origin main
   ```

2. **Vercel.** Зайдите на vercel.com → **Add New → Project** → импортируйте репозиторий.
   Vercel сам определит Next.js. Настройки менять не нужно — жмите **Deploy**.

3. **Домен.** В проекте на Vercel: **Settings → Domains → Add** → `sitkaz.kz`.
   Vercel покажет DNS-записи (A / CNAME) — пропишите их у регистратора домена.
   После проверки DNS сайт откроется на sitkaz.kz.

## Telegram Mini App (опционально, позже)

1. В **@BotFather** создайте бота, затем `/newapp` → привяжите Mini App к URL `https://sitkaz.kz`.
2. Приложение уже подключает `telegram-web-app.js` и вызывает `expand()` — внутри Telegram
   развернётся на весь экран автоматически.
